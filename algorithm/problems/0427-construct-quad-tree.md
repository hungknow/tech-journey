# 0427 Construct Quad Tree

## Problem Description

Given a `n x n` matrix `grid` consisting of values `0` and `1`, return the quad tree representing the same grid.

A quad tree is a tree data structure where each internal node has exactly four children.

Each node of the quad tree has:
- `val`: `True` if the node represents a grid of all 1's or `False` if all 0's
- `isLeaf`: `True` if the node is a leaf node
- `topLeft`, `topRight`, `bottomLeft`, `bottomRight`: child nodes
